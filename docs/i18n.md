# 多語言支持 (i18n)

## 概述

系統支持繁體中文和英文，基於 react-i18next 框架實現。

## 配置結構

```
frontend/src/i18n/
├── index.ts        # i18n 配置入口
└── locales/
    ├── zh.json     # 繁體中文
    └── en.json     # 英文
```

## 使用方式

### React 組件

```tsx
import { useTranslation } from 'react-i18next';

function PortfolioPage() {
  const { t } = useTranslation();
  
  return (
    <div>
      <h1>{t('portfolio.title')}</h1>
      <button>{t('portfolio.create')}</button>
    </div>
  );
}
```

### 語言切換

```tsx
import { useTranslation } from 'react-i18next';

function LanguageSwitcher() {
  const { i18n } = useTranslation();
  
  const changeLanguage = (lang: string) => {
    i18n.changeLanguage(lang);
  };
  
  return (
    <div>
      <button onClick={() => changeLanguage('zh')}>繁體中文</button>
      <button onClick={() => changeLanguage('en')}>English</button>
    </div>
  );
}
```

## 命名規範

- **命名空間**: 按功能模塊劃分 (common, portfolio, alert, etc.)
- **鍵名**: 使用 snake_case
- **嵌套**: 最多三層 (namespace.section.key)

## 新增翻譯步驟

1. 在 `zh.json` 添加中文文案
2. 在 `en.json` 添加對應英文
3. 在組件中使用 `t('key.path')`

## 相關模塊

- 所有前端模塊均使用 i18n
